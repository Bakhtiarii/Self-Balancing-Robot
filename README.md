# Self-Balancing-Robot
Implement Self-Balancing robot by Arduino - UT EE - Summer 2019

The robot's purpose is to try to keep itself up simply.
# Required:
- Arduino Uno
- 2x Stepper motor driver
- 2x Stepper motor
- Gyroscope
- Voltage regulator
- Battery
- 2x Tires
- Chassis
# Optional:
- Bluetooth module
- SelfBalancing application
